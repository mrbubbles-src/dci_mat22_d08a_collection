Schreiben Sie den passenden CSS Code im entsprechenden Teil der gegebenen Datein(entweder in der index.html oder style.css).
Orientieren Sie sich bei der Gestaltung am Bild Vorgabe.png !
Schreiben Sie zuerst das korrekte HTML mit den richtigen TAG Elemente!
Lösung Sie die Aufgaben mit puren CSS!

Hier die genauen Aufgaben:
  1. Ändern sie bei dem ersten div Element die Schriftgröße, Schrift Familie, Schriftstil und alles soll in Großbuchstaben geschrieben sein
  2. Ändern sie das Tag Element Paragraph(Klasse erstes-p) die Text Dekoration die Schriftgröße und den Abstand zwischen den einzelnen Buchstaben
  3. Gestalten sie den Button mit einem Border, Border Umrandung und mit weiteren Farben
  4. Bei einem Hover soll der Button die Farben und die Größe ändern
  5. Die gegebenen Navigationsleiste soll nebeneinander dargestellt werden, der List-Style gelöscht werden, die Farben sollen angepasst werden und der unterstrichene Text soll gelöscht werden
  6. Fügen Sie für den Hover Effekt bei den einzelnen Navigations Elementen eine andere Hintergrundfarbe hinzu
  7. Die Tabelle soll Umrandungen an allen Ecken bekommen und der Text zentriert werden. Zusätzlich soll der Kopf der Tabelle eine andere Hintergrundfarbe bekommen.
  8. Die Tabelle und die Navigationsleiste sollen zentral auf der Webseite dargestellt werden
